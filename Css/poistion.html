Static Position

This is static position. It is default position of all elements. It is not affected by top, bottom, left and right properties.

Relative Position

This is relative position. It is affected by top, bottom, left and right properties. It is positioned relative to its normal position.

Absolute Position

This is absolute position. It is affected by top, bottom, left and right properties. It is positioned relative to the nearest positioned ancestor (not static). If there is no positioned ancestor, it uses the document body, and moves along with page scrolling.
